# Workforce Salary Dynamics Analysis

![](https://github.com/Cuteod/Workforce-Salary-Dynamics-Analysis/blob/main/workforce.jpg)

A data analytics project exploring how employee salaries are influenced by gender, education level, years of experience, age, and seniority level within an IT workforce dataset.
This project uses Microsoft Excel to uncover compensation trends, identify pay gaps, evaluate career progression speed, and generate strategic HR insights through an interactive dashboard.

## Project Objective

The goal of this project was to answer key workforce questions such as:

- Does gender influence salary levels?
- How much does education impact earnings?
- Which employees progress faster into senior roles?
- How does salary grow with years of experience?
- Where do salary inequalities exist?

## Dataset Features

The dataset contains the following columns:

Age, Age Band, Gender, Education Level, Job Title, Years of Experience, Salary

Derived Columns:

Seniority Level, Salary Band, Salary per Year of Experience

## Key Insights

![](https://github.com/Cuteod/Workforce-Salary-Dynamics-Analysis/blob/main/Workforce%20insights.png)


1. Gender Gap Exists Early

The largest pay gap occurs at entry level, indicating possible hiring bias or inconsistent starting salaries.

2. Education Drives Income

Higher education significantly improves entry pay, progression speed, and salary ceiling.

3. Experience Alone Is Not Enough

At executive level, Bachelor's and Master's salaries converge, proving that leadership output matters more than credentials alone.

4. High Performers Plateau Later

Salary growth slows after age 45, meaning retention strategies should shift beyond salary alone.

## Conclusion

This project demonstrates how workforce data can be transformed into strategic insights that support better compensation planning, talent retention, and organizational fairness.
